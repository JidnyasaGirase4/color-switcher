A simple mini project that switches the page background color on click. Built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** [https://fantastic-strudel-339685.netlify.app/](https://fantastic-strudel-339685.netlify.app/)

## ✨ Features

- Generate a random background color with one click
- Apply a custom color of your choice
- Clean, minimal UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Getting Started

```bash
git clone https://github.com/JidnyasaGirase4/color-switcher.git
cd color-switcher
# Open index.html in your browser
```
